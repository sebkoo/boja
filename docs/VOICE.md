3. Any derived copy elsewhere in the repo must trace back to a line in this file.

## The feeling (tone bar)

> When someone confirms a plan in Boja, I want them to feel **connected, cherished, and appreciated.**

These three words are a tone bar, not literal UI copy. Microcopy must evoke them through behavior — remembered preferences, effortless confirmation — and never state them outright.

## The name

Boja (보자) is plain Korean for "let's see each other," as in 언제 한번 보자 — "let's hang out sometime," the sentence this product exists to convert into a real plan. The canonical way to introduce the name, everywhere:

> **Boja (보자 — "let's hang out") — the AI concierge that turns it into a real plan.**
